COMPUTER SCIENCE AND SOFTWARE ENGINEERING SEMINAR

Hacking Python Bytecode for Fun And...Well, Maybe Just for Fun

Adam Hartz (Faculty candidate and presenter)
Massachusetts Institute of Technology

  • Date: Wednesday, 27 March 2019
  • Time: 4:00 PM
  • Room: Olin 267

 

When the Python interpreter executes a program, it does so by first converting the human-readable source code to an intermediate representation known as bytecode, which is the representation that is actually executed by the interpreter. In this talk, I'll provide a brief introduction to Python bytecode and how it is interpreted, and I'll also demonstrate examples of analyzing and/or manipulating this representation in order to better understand (or modify!) details of Python's interpretation of the code.

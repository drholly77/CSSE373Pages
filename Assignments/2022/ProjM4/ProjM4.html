1. ProjM4 Statement of Work

  • Implement UnlabeledUndirectedGraphChecking1
  • It must:
    • be implemented as shown in the Checking Component Construction instructional materials (see References in Section 0 above)
    • be stored in the folder: components\include\UnlabeledUndirectedGraph
  • You may want to use the Graph Driver and your UnlabeledUndirectedGraph1.hpp to test your checking component
  • Note:
    • The Graph Driver is implemented so as to stop a keyboard user from violating some (possibly all) of UnlabeledUndirectedGraph1's operation's preconditions
    • You may need to alter Graph Driver's code so that it allows the keyboard user to deliberately violate a precondition - you might consider creating an entire new project called NakedGraphDriver, where this driver has had all protection stripped from it, allowing a malicious keyboard user to execute commands that violate the UnlabeledUndirectedGraph1's operation's preconditions
    • This will allow you to test your implementation of UnlabeledUndirectedGraphChecking1

2. Grading of ProjM4

  • Your UnlabeledUndirectedGraphChecking1.hpp will be compiled/built/tested in Debug Mode with Dr. Holly's private set of tests or a modified Graph Driver
  • These tests will attempt to call an UnlabeledUndirectedGraph1 operation with a call that violates its non-trivial precondition

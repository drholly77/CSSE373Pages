Three steps to getting your Unit Test project up and running

This file contains instructions (below) for following 3 steps:

  1. Create a new VS2022 Unit Test project
  2. Add the Wrapper Filter and Wrapper Files
  3. Update the project's settings

Create a new VS2022 Unit Test project

Step 2: Add the Wrapper Filter and Wrapper Files

  1. In the L2 Solution, right click the Source Files folder icon and choose Add | New Filter ...
  2. Name the new filter wrapper
  3. Now, right click the wrapper filter (looks like a folder) and choose Add | Add Existing Item ...
    We're going to add files to the wrapper filter.
  1. From the "Add Existing Item" dialog you must navigate to the C:\CppDev\wrapper\class folder
    (see screen shot to right).
  2. Select all the .cpp files that appear in the C:\CppDev\wrapper\class folder, then click OK

How to Turn in a Lab Assignment

On or before the Lab assignment's appointed due date (see Moodle assignment for exact due date) turn in your Lab X by doing the three steps listed below.

Note: substitute L1, L2, etc., for LX in the following. For example, if you are working on Lab 1, substitute L1 for LX.

 

1. While in Visual Studio working on LX, do the following:

  1. In Debug configuration, run the Clean operation.  From the menu bar choose: Build | Clean Solution
  2. Now switch to Release configuration and run the Clean operation

2. Quiz Visual Studio and open up Windows File Explorer and do the following:

3. Create a .zip of your assignment

Navigate to the C:\CppDev folder, and right click on the LX folder, and choose:
Send to | Compressed (zipped) Folder

This will create the file: LX.zip (it should be approximately 100 KB or less in size)
See screen shot to the right

4. Upload the .zip to the Moodle assignmet

Upload the .zip created in Step 3 (above)

Changing Project's Settings to use Wrapper Libraries

There are two steps required when changing the project's settings to link to the WrapperD.lib and WrapperR.lib:

  1. Tell linker to search an additional folder for .lib files
  2. Tell the linker to link to WrapperD.lib in Debug mode and to WrapperR.lib in Release mode

1) C343 uses C++ and Microsoft Visual Studio 2015 for the lab assignments.

  • C343 will be using Microsoft's Visual Studio Community 2015 this semester for our C++ development environment.
    • Note: You must use Visual Studio 2015. You will be supplied with libraries compiled by VS 2015, and your lab submissions will be compiled for grading by the grader using VS 2015.   • For this lab assignment:
    1. Download (for free) Visual Studio Community 2015 from Microsoft's DreamSpark web site.
    2. Install Visual Studio Community 2015 on the machine you'll be working on this fall semester.
  • IU has a Knowledge Base entry on how to use DreamSpark (http://kb.iu.edu/data/axjl.html)
  • The download (from #1 above) comes in an .iso file, which means you'll need to burn it to a DVD.
  • Then install Visual Studio from the DVD.

2) Setup CppDevFa15 Folder

Overview
  • The CppDevFa15 folder contains software components that you will be using this semester when writing your software and it is where you will store all your C343 software development lab assignments.
    • Note: versions of this folder prior to CppDevFa15 will fail to work correctly, do not use any of these versions.
  • Place this folder on your computer's C:\ drive, and work exclusively from this folder all semester.  (Recommended)
  • Alternatives to working exclusively from your computer's C:\ drive:
    1. Work from your C:\ drive, but when you need to work on another computer, copy the CppDevFa15 folder from your C:\ drive to a USB flash drive. Temporarily work from the flash drive, then when done, copy the CppDevFa15 folder back onto your C:\ drive.
    2. If you are going to use multiple computers, you might choose to work exclusively from a USB flash drive. With this option, you will work on all you software development labs from this flash drive. The drawback with this option is that all file manipulations will be much slower including compiling, editing, and saving.
Instructions
  1. To download, click on the following zip file and save it to your computer's hard drive: CppDevFa15.zip
  2. Once it is downloaded, navigate to inside the .zip by double clicking on it.
  3. Once inside the zip, copy the folder found inside (called "CppDevFa15") to your C:\ drive.
  4. After it is placed onto your C:\ drive, it should look like the screen shot to the right.

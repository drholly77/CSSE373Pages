How to Turn in a Project Assignment

On or before the Project assignment's appointed due date (see Canvas assignment for exact due date) turn in your Project X by doing the three steps listed below.

Note: substitute P1, P2, etc., for PX in the following. For example, if you are working on Project 1, substitute P1 for PX.

  1. While in Visual Studio working on PX, do the following:
    1. In Debug configuration, run the Clean operation.  From the menu bar choose: Build | Clean Solution
    2. Now switch to Release configuration and run the Clean operation
  2. Switch from Visual Studio to Windows File Explorer to do the following:
    1. Navigate to the C:\CppDev\PX folder and if either of the following files are present delete them: "PX.VX.db" or "PX.sdf"
      This file is large and will slow down uploading and downloading. Visual Studio creates this file automatically, and will easily recreate it when it needs it
    2. Navigate to the C:\CppDev folder, and right click on the PX folder, and choose:
      Send to | Compressed (zipped) Folder
      This will create the file: PX.zip (it should be approximately 100 KB or less in size)
      See screen shot to the right
  3. Finally, you must upload the .zip created in Step 2 (above). To do this, log into Canvas, navigate to the PX Assignment, click "Submit Assignment", and follow the on screen instructions in order to upload you PX.zip file

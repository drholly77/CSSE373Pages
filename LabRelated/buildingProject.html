Building a Project

After creating a new project solution, adding files, and changing settings, you are now ready to build the project, which causes all the .cpp files to be compiled.

  • Compiling the entire project -
    • From the menu bar choose: Build | Build Lab
    • If you always use the Build LX.exe command, Visual C++ will only compile those files that have changed since the last time you compiled the project. This will save you time.
  • The executable file, (i.e., the .EXE) will be located in the DEBUG subdirectory.
  • You can also compile just one file at a time.
    • To compile one file at a time (instead of the entire project), from the Solution Explorer window right click on the .CPP file that needs compiling, choose "Compile" from the quick menu.

1.2 Applications of Propositional Logic to Logal Systems

 Definitions

  • consistent - when specifications do not contain conflicts that could lead to contradictions
  • conjunction - a compound proposition using ∧
  • conjunct - an operand to the ∧ operator

Example 1 - Are the following statements consistent? That is, are all True at the same time?

  1. "It snowed or I went to school"
  2. "It did not snow"
  3. "If snowed, then I went to school"

Step 1)

Represent as logic using propositional variables p and q

p: "It snowed"

q: "I went to school"

Rewrite the English propositions using propositional variables and logical operators:

1) p ∨ q          "It snowed or I went to school"

2) ~p             "It did not snow"

3) p → q         "If snowed, then I went to school"

Step 2)

Now, create one big compound proposition from lines 1, 2, and 3, using the AND logical operator, name this proposition b (for big)

b: (p ∨ q) ∧ ~p ∧ (p → q)

The three things AND'ed together in b are each called a conjunct.

In order for b to be consistent, b has to evaluate to True. In order for b to evaluate to True, each conjunct must evaluate to True

Step 3)

Try to find an assignment of True and False to the two propositional variable (p and q) used in proposition b, such that b evaluates to True

3.1) Start with the conjunct ~p -
In order for ~p to evaluate to True, p has to be False.
So we have an assignment for p, now we must come up with an assignment for q

3.2) Work with the first conjunct (p ∨ q), substitute our assignment for p (which is False) into this proposition and get (False ∨ q).
In order for this proposition to evaluate to True, q must have the assignment of True.
That would give (False v True) which evaluates to True.
So now we have q with the assignment of True

3.3) With p = False and q = True, verify that the 3rd conjunct (p → q) evaluates to True.
Substitute p and q's values and get (False → True).
Check Implication's Truth Table to determine if this evaluates to True.
The Truth Table says it does, so the 3rd conjunct evaluates to True when p = False and q = True

3.4) Wrap it up. With p = False and q = True, all 3 of b's conjuncts evaluate to True, so this System is consistent, i.e., it is not a contradication.
